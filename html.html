Elementen

  • <h1> - <h2> - <h3> - ... elementen kun je gebruiken voor titels.
  • het <p> element wordt gebruikt voor paragraaf elementen.
  • <em> elementen worden gebruikt om iets te beklemtonen teskt wordt schuin weer gegeven.
  • Elementen kunnen vetgedrukt worden met het <strong> element.
  • <img scr="" alt=""> opbouw voor foto toe te voegen
  • <a href="">link naar andere website</a> , ander gedeelte van de website...
  • Alles in <body> wordt weergegeven op de websitepagina
  • Opbouw van een website gebeurd in dit formaat: header - main - footer
  • Commentaar schrijven in HTML kan met behulp van "<!-- -->"
  • Elementen worden opgebouwd door ze te openen "< open >" "<sluiten />"
  • 7 januari <time datetime="YYYY-MM-DD">DD MM</time> opbouwing.

index.html

index.html zonder hoofdletter!

  • <!DOCTYPE html> //Zegt aan browser welk document type de rest van het document is.//
  • <html lang="nl"> //taal - veranderen naar de taal dat je op je website gebruikt. Standaard "en" dus op letten//
  • <head> bevattend:
    • <meta charset="UTF-8">
    • <meta name="viewport" content="width=device-width, initial-scale=1.0">
    • <title>Document< /title>
    • < link rel="stylesheet" href="file.css"> //Opbouwing link voor CSS bestand.//
    • <script src="file.js" defer></script> //Opbouwing link voor JavaScript bestand.//
    • </head> als sluiting

Alles binnen het body gedeelte wordt weergegeven op de pagina. We gebruiken <body> als opening en </body> als sluiting.

Om ons html bestand te sluiten gebruiken we </html>

Lijsten

Er zijn verschillende elementen om een opsomming te maken:

  • <ul> unorderd list
    • volgorde onbelangrijk
    • Binnen deze lijst wordt het element <li> toegevoegd.
  • <ol> ordend list
    • Volgorde belangrijk
    • Binnen deze lijst wordt het element <li> toegevoegd.
  • <dl> discription list
      • Binnen deze lijst wordt het element <dt> EN <dd> toegevoegd.
